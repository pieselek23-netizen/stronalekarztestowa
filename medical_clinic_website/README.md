# Gabinet Lekarski - Nowoczesna Opieka Medyczna

Profesjonalna strona internetowa dla prywatnego gabinetu lekarskiego. Nowoczesny design, responsywna, łatwa w obsłudze.

## 🎯 Cechy

- ✅ **Responsywny design** – Działa na wszystkich urządzeniach (mobile, tablet, desktop)
- ✅ **Minimalistyczny design** – Profesjonalny wygląd z kolorystyką: biały, szary, niebieski
- ✅ **Szybki** – Zoptymalizowany dla szybkiego ładowania
- ✅ **SEO-friendly** – Gotowy do wyszukiwarek
- ✅ **Dostępny** – Zgodny ze standardami dostępności

## 📋 Sekcje Strony

1. **Header** – Nawigacja z logo i menu mobilne
2. **Hero Section** – Hasło "Nowoczesna opieka medyczna" z CTA
3. **O nas** – Informacje o lekarzu i doświadczeniu
4. **Usługi** – 6 usług medycznych z ikonami
5. **Opinie** – Recenzje pacjentów z gwiazdkami
6. **Rezerwacja** – Formularz do umówienia wizyty
7. **Kontakt** – Dane kontaktowe i mapa Google
8. **Footer** – Linki i informacje

## 🛠️ Technologia

- **React 19** – Framework do budowy UI
- **Tailwind CSS 4** – Stylizacja
- **Vite** – Build tool
- **TypeScript** – Typowanie
- **Lucide React** – Ikony
- **Google Maps API** – Mapa lokalizacji

## 🚀 Wdrożenie na GitHub Pages

### Wymagania
- Git zainstalowany
- Konto GitHub

### Kroki

1. **Utwórz repozytorium na GitHub**
   - Przejdź na https://github.com/new
   - Nazwa: `medical-clinic-website`
   - Wybierz "Public"

2. **Sklonuj repozytorium**
   ```bash
   git clone https://github.com/TWOJA_NAZWA/medical-clinic-website.git
   cd medical-clinic-website
   ```

3. **Skopiuj pliki strony**
   - Rozpakuj archiwum `medical_clinic_netlify.zip`
   - Skopiuj wszystkie pliki do folderu repozytorium

4. **Wyślij na GitHub**
   ```bash
   git add .
   git commit -m "Initial commit: Medical clinic website"
   git push -u origin main
   ```

5. **Włącz GitHub Pages**
   - Przejdź do Settings → Pages
   - Source: `main` / `/ (root)`
   - Kliknij Save

6. **Czekaj na deployment**
   - GitHub wdroży stronę automatycznie
   - Strona będzie dostępna pod: `https://TWOJA_NAZWA.github.io/medical-clinic-website/`
